Preguntas y Respuestas del Laboratorio 3

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Sirve para que un atributo CSS tenga prevalencia y prioridad ante otras instucciones, solo lo usaría cuando el atributo CSS es necesario en declaraciones posteriores.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Esta puede dificultar la presentación de la información en la página, por eso se tiene que elegir un fondo con mucho cuidado.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de una propiedad de estilo entre %, px y pt?

Yo prefiero usar pixeles, pero todo depende de las especificaciones del proyecto o tarea la cual estes realizando.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Se hace para reducir el tamaño del archivo raw de CSS, haciendo que el rendimiento del sito web sea mejor.
