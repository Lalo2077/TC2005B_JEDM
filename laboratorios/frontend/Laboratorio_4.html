Preguntas y Respuestas del Laboratorio 4

¿Que diferencias y semejanzas hay entre Java y Javascript?

Diferencias: Javascript esta enfocado en la interacción web y es interpretado. Java es un lenguaje orientado a objetos y es compilado.
Semejanzas: Su sintaxis inspirada en C y el nombre de Java.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getMinutes: devuelve los minutos
getHours: devuelve las horas
getDay: devuelve los días de la semana
getFullYear: devuelve el año completo
getDate: devuele los días del mes

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

ToString: vuelve a un arreglo a un string
push: agrega un elemento al final del arreglo
splice: inserta o elimina elementos del arreglo
pop: elimina el último elemento agregado
concat: se usa para unir dos o más arreglos
